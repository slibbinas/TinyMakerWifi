Boot animation

Pick which animation plays at power-on and press Save config. Show plays it on the printer's screen (idle only) - check the printer. Delete removes the file from the SD card.

Backup & restore?

Requires TinyMaker Connect registration.
The backup holds every setting and the lifetime counters. With a backup on the SD card, the printer offers to restore it on the first boot after a full USB reflash.
Config locks automatically while printing.
